## Worum geht es?

Dieses Repository dokumentiert die Strukturen, Personen und Netzwerke rund um:

- **TNHCO** (Terra Nova Helvetica Genossenschaft, Schweiz)
- **GoldenEra** (goldenera.global — SunCoin ICO, „White Spiritual Boy“-Kult)
- **Monetary One** („Program Sunrise“, Treasury-Plattform)
- **WSBOH** u.a. verbundene Entitäten

Die Recherche verfolgt das Ziel, ein möglichst vollständiges Bild der Akteure,
Firmengeflechte und öffentlichen Behauptungen zu erstellen — **ausschließlich
auf Basis öffentlich zugänglicher Informationen.**
